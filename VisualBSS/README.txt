	-Qt 5.7 or sup
	-OpenGL 3.3 ou sup

	In Pdp_Velos repository:
		-mkdir build && cd build
		-qmake ../Pdp_Velos.pro
		-make

===============================================================================
Execution:
	./Pdp_Velos
